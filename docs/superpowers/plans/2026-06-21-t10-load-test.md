# T10 Contention Load Test — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Measure single-creator `execute_split` throughput/latency under a concurrent burst (isolated to shared-vault contention) and decide the spec §15 v1 mitigation.

**Architecture:** A tsx harness (`scripts/t10-load-test.mts`) signs from one testnet key. Per tier N, it pre-splits owned SUI gas coins + USDC input coins, **pre-builds** all N `execute_split` tx bytes, then fires them concurrently — the only shared contention is `SplitConfig` + `TaxVault` + `SavingsVault`. A pure helper module (`scripts/t10-lib.ts`) does stats / abort-classification / conservation math and is unit-tested with vitest.

**Tech Stack:** `@mysten/sui` 2.19.0 (`SuiGrpcClient` from `@mysten/sui/grpc`, `Ed25519Keypair`), tsx, vitest. Reuses `buildExecuteSplit` + IDs from `src/lib/`.

## Global Constraints

- SDK: `@mysten/sui` **2.19.0**; gRPC execution path (JSON-RPC/Quorum Driver disabled, Protocol 124). `Transaction` (not `TransactionBlock`).
- Signer key from env `SUI_PRIVATE_KEY` (bech32 `suiprivkey…`) — **never hardcoded, never committed**.
- Config/vault IDs + owner address from env (`T10_CONFIG_ID`, `T10_OWNER`) with constants.ts fallback — re-runnable.
- `AMOUNT_RAW = 10000` (0.01 USDC), `FIXED_GAS_BUDGET = 20_000_000n` MIST.
- Tiers run smallest→largest: `[10, 50, 150, 350]`.
- **No auto-retry** in the burst (retry masks the throughput ceiling).
- Burst must time only sign+execute — all `tx.build()` happens in prep.

// web/creatorflow-web/scripts/t10-lib.ts
export function percentile(sortedAsc: number[], p: number): number {
  if (sortedAsc.length === 0) return NaN;
  // nearest-rank: rank = ceil(p/100 * N), clamped to [1, N]
  const rank = Math.min(sortedAsc.length, Math.max(1, Math.ceil((p / 100) * sortedAsc.length)));
  return sortedAsc[rank - 1];
}

export type Bucket = "success" | "congestion" | "locked" | "terminal" | "network";

export function classify(r: { ok: true } | { ok: false; error: string }): Bucket {
  if (r.ok) return "success";
  const e = r.error;
  if (/congest/i.test(e)) return "congestion";
  if (/lock|equivocat|not available for consumption/i.test(e)) return "locked";
  if (/moveabort|abort|^e[A-Z]|status/i.test(e)) return "terminal";
  if (/fetch|econn|timeout|network|socket/i.test(e)) return "network";
  return "terminal";
}

export function conserves(
  parts: { payouts: bigint[]; tax: bigint; savings: bigint; fee: bigint },
  amountIn: bigint,
): boolean {
  const sum = parts.payouts.reduce((a, b) => a + b, 0n) + parts.tax + parts.savings + parts.fee;
  return sum === amountIn;
}
```

// web/creatorflow-web/scripts/t10-load-test.mts
/**
 * T10 contention load test (spec docs/superpowers/specs/2026-06-21-t10-load-test-design.md).
 * Single-key N-coin fan on testnet: isolates shared TaxVault+SavingsVault serialization.
 *
 * Run: cd web/creatorflow-web && \
 *   SUI_PRIVATE_KEY=suiprivkey... T10_CONFIG_ID=0x... \
 *   npx -y tsx scripts/t10-load-test.mts [tier]
 *   (no [tier] arg → runs 10,50,150,350; a number → runs just that tier)
 */
import { SuiGrpcClient } from "@mysten/sui/grpc";
import { Transaction } from "@mysten/sui/transactions";
import { Ed25519Keypair } from "@mysten/sui/keypairs/ed25519";
import { buildExecuteSplit } from "../src/lib/ptb";
import { PACKAGE_ID, NETWORK, USDC_TYPE } from "../src/lib/constants";
import { percentile, classify, conserves, type Bucket } from "./t10-lib";

const AMOUNT_RAW = 10_000n;
const FIXED_GAS_BUDGET = 20_000_000n;
const TIERS = [10, 50, 150, 350];
const BASE_URL = "https://rpc.testnet.sui.io";

const PK = process.env.SUI_PRIVATE_KEY;
if (!PK) throw new Error("SUI_PRIVATE_KEY (suiprivkey… bech32) required");
const signer = Ed25519Keypair.fromSecretKey(PK);
const OWNER = process.env.T10_OWNER ?? signer.toSuiAddress();
const CONFIG_ID = process.env.T10_CONFIG_ID;
if (!CONFIG_ID) throw new Error("T10_CONFIG_ID required");

const client = new SuiGrpcClient({ network: NETWORK, baseUrl: BASE_URL });

type Json = Record<string, unknown>;

/** Read config object → version + vault ids. */
async function readConfig(): Promise<{ version: bigint; taxVaultId: string; savingsVaultId: string }> {
  const { object } = await client.getObject({ objectId: CONFIG_ID!, include: { json: true } });
  const j = object?.json as Json | undefined;
  if (!j) throw new Error(`config ${CONFIG_ID} has no json content`);
  const version = BigInt(String(j.version));
  const taxVaultId = String(j.tax_vault_id);
  const savingsVaultId = String(j.savings_vault_id);
  if (!/^0x[0-9a-f]{64}$/.test(taxVaultId) || !/^0x[0-9a-f]{64}$/.test(savingsVaultId))
    throw new Error(`config missing tax/savings vault ids: ${taxVaultId} ${savingsVaultId}`);
  return { version, taxVaultId, savingsVaultId };
}

type ObjRef = { objectId: string; version: string; digest: string };

/** Split one source coin (USDC or gas SUI) into `n` coins of `each`, transfer back to owner.
 *  Returns full refs for the n created coins (gas payment needs version+digest). Awaits finality. */
async function fanCoins(kind: "usdc" | "gas", n: number, each: bigint): Promise<ObjRef[]> {
  const tx = new Transaction();
  tx.setSender(OWNER);
  let source;
  if (kind === "usdc") {
    const coins = await client.getCoins({ owner: OWNER, coinType: USDC_TYPE });
    const big = coins.data.find((c) => BigInt(c.balance) >= each * BigInt(n));
    if (!big) throw new Error(`no single USDC coin ≥ ${each * BigInt(n)}; merge first`);
    source = tx.object(big.coinObjectId);
  } else {
    source = tx.gas; // split the gas coin itself
  }
  const parts = tx.splitCoins(source, Array.from({ length: n }, () => tx.pure.u64(each)));
  tx.transferObjects(Array.from({ length: n }, (_, i) => parts[i]), tx.pure.address(OWNER));
  const res = await client.signAndExecuteTransaction({ transaction: tx, signer });
  if (res.$kind !== "Transaction") throw new Error(`fan ${kind} failed: ${JSON.stringify(res)}`);
  await client.waitForTransaction({ digest: res.Transaction.digest });
  // Created coin refs straight from effects — objectId + post-tx version + digest.
  const created: ObjRef[] = (res.Transaction.effects?.changedObjects ?? [])
    .filter((o: Json) => String((o as Json).idOperation) === "Created")
    .map((o: Json) => ({
      objectId: String((o as Json).objectId),
      version: String((o as Json).outputVersion ?? (o as Json).version),
      digest: String((o as Json).outputDigest ?? (o as Json).digest),
    }))
    .filter((r) => /^0x[0-9a-f]{64}$/.test(r.objectId) && r.version !== "undefined" && r.digest !== "undefined");
  if (created.length < n) throw new Error(`expected ≥${n} created ${kind} coins, got ${created.length}`);
  return created.slice(0, n);
}

type TierResult = {
  n: number; wallMs: number; tps: number;
  p50: number; p90: number; p99: number;
  buckets: Record<Bucket, number>;
};

async function runTier(n: number, cfg: Awaited<ReturnType<typeof readConfig>>): Promise<TierResult> {
  console.log(`\n=== tier N=${n} ===`);
  const usdcCoins = await fanCoins("usdc", n, AMOUNT_RAW);
  const gasCoins = await fanCoins("gas", n, FIXED_GAS_BUDGET * 2n);

  // PRE-BUILD all N tx bytes (object resolution happens here, OUT of the timed burst).
  const built: Uint8Array[] = [];
  for (let i = 0; i < n; i++) {
    const tx = buildExecuteSplit({
      configId: CONFIG_ID!, taxVaultId: cfg.taxVaultId, savingsVaultId: cfg.savingsVaultId,
      amountIn: AMOUNT_RAW, expectedVersion: cfg.version, usdcCoinIds: [usdcCoins[i].objectId],
    });
    tx.setSender(OWNER);
    tx.setGasPayment([gasCoins[i]]); // full {objectId,version,digest} ref → no build-time resolve
    tx.setGasBudget(FIXED_GAS_BUDGET);
    built.push(await tx.build({ client }));
  }

  // BURST: time only sign + execute.
  const buckets: Record<Bucket, number> = { success: 0, congestion: 0, locked: 0, terminal: 0, network: 0 };
  const lat: number[] = [];
  const t0 = performance.now();
  const settled = await Promise.allSettled(built.map(async (bytes) => {
    const start = performance.now();
    try {
      const { signature } = await signer.signTransaction(bytes);
      const res = await client.executeTransaction({ transaction: bytes, signatures: [signature] });
      const ok = res.$kind === "Transaction";
      return { ms: performance.now() - start, r: ok ? { ok: true as const } : { ok: false as const, error: JSON.stringify(res) } };
    } catch (e) {
      return { ms: performance.now() - start, r: { ok: false as const, error: String(e) } };
    }
  }));
  const wallMs = performance.now() - t0;
  for (const s of settled) {
    if (s.status !== "fulfilled") { buckets.network++; continue; }
    lat.push(s.value.ms);
    buckets[classify(s.value.r)]++;
  }
  lat.sort((a, b) => a - b);
  const out: TierResult = {
    n, wallMs, tps: (buckets.success / wallMs) * 1000,
    p50: percentile(lat, 50), p90: percentile(lat, 90), p99: percentile(lat, 99), buckets,
  };
  console.log(`  wall=${wallMs.toFixed(0)}ms tps=${out.tps.toFixed(1)} p50=${out.p50.toFixed(0)} p90=${out.p90.toFixed(0)} p99=${out.p99.toFixed(0)}`);
  console.log(`  buckets=${JSON.stringify(buckets)}`);
  return out;
}

function table(rows: TierResult[]): string {
  const h = "| N | wall(ms) | TPS | p50 | p90 | p99 | success | congestion | locked | terminal | network |";
  const sep = "|---|---|---|---|---|---|---|---|---|---|---|";
  const body = rows.map((r) =>
    `| ${r.n} | ${r.wallMs.toFixed(0)} | ${r.tps.toFixed(1)} | ${r.p50.toFixed(0)} | ${r.p90.toFixed(0)} | ${r.p99.toFixed(0)} | ${r.buckets.success} | ${r.buckets.congestion} | ${r.buckets.locked} | ${r.buckets.terminal} | ${r.buckets.network} |`);
  return [h, sep, ...body].join("\n");
}

async function main() {
  const arg = process.argv[2];
  const tiers = arg ? [Number(arg)] : TIERS;
  const cfg = await readConfig();
  console.log(`owner ${OWNER.slice(0, 14)}… config ${CONFIG_ID!.slice(0, 14)}… version ${cfg.version}`);
  console.log(`tax ${cfg.taxVaultId.slice(0, 14)}… savings ${cfg.savingsVaultId.slice(0, 14)}…`);
  const rows: TierResult[] = [];
  for (const n of tiers) rows.push(await runTier(n, cfg));
  console.log("\n" + table(rows));
  console.log("\nNOTE: latency resolve = validator execution-ack, not checkpoint finality.");
  console.log("conserves() sample check: split math is enforced on-chain (Move tests); congestion>0 with success>0 = serialized-but-throughput-bound (expected T10 signal).");
}

main().catch((e) => { console.error(e); process.exit(1); });
```

## Notes for the implementer

- The harness hits **live testnet** and spends real gas + USDC. It is not idempotent;
  each run consumes `Σ tiers × 0.01` USDC and gas. Smoke at N=50 before the full run.
- `fanCoins("gas", …)` splits `tx.gas`; the SDK still needs a gas coin to pay for that
  split tx itself — leave the largest SUI coin unsplit as the fan tx's own gas, or run
  the gas fan from a separately-funded coin. If the split tx fails for gas, merge SUI
  first (`sui client merge-coin`).
- If `getObject` json field names differ (`tax_vault_id` vs camelCase), match the actual
  on-chain field names already used by `scripts/cap-defense-demo.mts` / the app's
  `src/lib/vault.ts` (snake_case confirmed in progress notes).
